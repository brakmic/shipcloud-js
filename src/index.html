Welcome to ShipCloud API Bindings for JavaScript
Open your browser console and initialize the API client with:
var api = new ShipCloudApi('YOUR_API_KEY');
You'll likely run into CORS exceptions when trying to communicate with api.shipcloud.io directly. This is due to certain HTTP restrictions that constrain cross-domain communication.
One possible solution would be a proxy server based on HapiJS or Express to forward local queries.
